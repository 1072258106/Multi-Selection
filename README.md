##Requirements
- Android SDK 17+

##Usage

Add to your root build.gradle:
```Groovy
allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

Add the dependency:
```Groovy
dependencies {
	        compile 'com.github.yalantis:multi-selection:v0.1'
}
```
